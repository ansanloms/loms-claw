以下の 4 カテゴリについて、Web 検索で今日のニュース・情報を収集し報告しろ。

## カテゴリ

### 1. ☀ 今日の天気（東京）

天気・気温（最高/最低）・降水確率を簡潔にまとめろ。

優先ソース:

- tenki.jp
- weathernews.jp

### 2. 🏍 バイク

バイク関連の最新ニュース・新車情報・イベント等から注目記事を 2〜3 件。

優先ソース: news.webike.net、www.autoby.jp、www.bikejin.jp

### 3. ⛰ 登山・山岳

登山・山岳関連の最新ニュース・山行情報・規制情報・遭難情報等から注目記事を 2〜3 件。

優先ソース:

- www.yamakei-online.com
- yamahack.com

### 4. 💻 技術ニュース

ソフトウェア・Web 技術・OSS 関連の注目記事を 3〜5 件。

優先ソース:

- Hacker News(news.ycombinator.com)
- www.publickey1.jp
- zenn.dev トレンド

## ソースの扱い

優先ソースに該当記事がなければ他のソースから探せ。優先ソースに固執するな。

## 出力形式

Discord マークダウンで以下の形式に従え。カテゴリ間は空行で区切る。

天気:

```
## ☀️ 今日の天気（東京）

（天気・気温・降水確率を地の文で記述）
```

その他のカテゴリ:

> `-# …` は discord 上でサブテキストとして表示される。

> URL を `<` と `>` で囲むと discord 上でプレビュー表示されなくなる。

```
## 🏍️ バイク

### 記事タイトル

記事の要約。300 文字程度で内容を伝えろ。

-# YYYY-MM-DD <URL>

（以下同様に繰り返す）
```

### ルール

- 要約は 1 記事あたり 300 文字程度。短すぎず、長すぎるな。
- URL は省略せず原文のまま貼れ。
- 情報が見つからないカテゴリは「該当なし」と書け。捏造するな。
