東京の今日の天気を報告しろ。

## 内容

天気・気温（最高/最低）・降水確率を簡潔にまとめろ。

優先ソース:

- tenki.jp
- weathernews.jp

## 出力形式

Discord マークダウンで以下の形式に従え。

```
## {天気アイコン} 今日の天気（東京）

（天気・気温・降水確率を地の文で記述）
```

天気アイコンの組み立て方:

単体の天気アイコン:
- ☀️ 晴れ
- ☁️ 曇り
- 🌧️ 雨
- ❄️ 雪
- ⛈️ 雷雨

「のち」の場合 `-` 、「時々」の場合 `/` で繋げろ。
例: 雨のち晴れ → 🌧️ - ☀️、曇り時々雨 → ☁️ / 🌧️、晴れのち曇り → ☀️ - ☁️

### ルール

- 前置き・挨拶・余計な文言は一切不要。いきなり本文から始めろ。
- 出力は 1 メッセージに収めろ。
