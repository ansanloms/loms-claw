# yamareco

ユーザの登山履歴・各山行のタイム/ペース、および登山計画 (予定行程) をヤマレコ (yamareco.com) から取得する skill。

## できること

- 登山計画の所要時間予測
- 過去実績からのペース倍率の集計
- ヤマレコに登録済みの計画の予定行程・距離・累積標高の確認

山行リスト・計画リストは API から、個別山行および個別計画の詳細は公開設定のもののみ HTML から抽出する。

## 対象外

- 非公開設定の個別山行・個別計画の詳細

## 発動する場面

登山計画の所要時間予測、過去実績からのペース倍率の集計、登録済み計画の予定行程・距離・累積標高の確認を求められたとき。

## 導入

```sh
apm install ansanloms/skills/yamareco --target claude
```

詳細は [SKILL.md](./SKILL.md) を参照。
