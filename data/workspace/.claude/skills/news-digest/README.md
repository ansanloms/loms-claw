# news-digest

指定テーマの最新ニュースを優先ソースから N 件集め、各記事をタイトル・要約・公開日・個別 URL の形にまとめて返す skill。

## できること

- 優先ソースに RSS/Atom があればフィードを先に取得、無ければ一般 Web 検索で補う
- 期間と除外リストでフィルタし、件数分を選んで各記事を要約
- 出力はタイトル・要約・公開日・個別 URL の中立フォーマット

## 対象外

- Discord 等への整形・投稿 (呼び出し側で行う前提。この skill には含めない)

## 発動する場面

「バイクの最新ニュースを集めて」「セキュリティ分野の注目記事を要約して」「最近の登山界隈の動きをまとめて」など、特定テーマの新着記事を収集して要約したいとき。

## 導入

```sh
apm install ansanloms/skills/news-digest --target claude
```
