# mountain-weather

登山対象の山の天気を、てんくらの登山指数 (A/B/C)・気象庁の麓の天気・Open-Meteo の山頂標高指定の数値予報・標高差からの推定で判断する skill。

## できること

- てんくらの登山指数 (A/B/C) の取得 (無料のサーバ HTML から)
- 気象庁の麓・登山口周辺の気象と併せた総合判断
- Open-Meteo の山頂標高指定の数値予報 (気温・風・降水・凍結高度) の取得 (無料の JSON API、CC BY 4.0)
- 標高差からの山頂気温の推定 (Open-Meteo が取得できない場合のフォールバック)

## 限界

山頂のピンポイント気象数値そのものは保証されない。指数・麓の天気・数値予報・推定で総合判断し、その限界をユーザに伝える前提。

## 発動する場面

「◯◯山の天気」「登山の天気」「明日山に登れるか」「登山指数」「山頂の気温」などを聞かれたとき。
