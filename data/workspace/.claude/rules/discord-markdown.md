# Discord メッセージのマークダウン

Discord に送信するメッセージで使える書式と使えない書式。

## 使える

- **太字** → `**太字**`
- *斜体* → `*斜体*` / `_斜体_`
- __下線__ → `__下線__`
- ~~取り消し線~~ → `~~取り消し線~~`
- ||スポイラー|| → `||スポイラー||`
- インラインコード → バッククォート1つで囲む
- コードブロック → バッククォート3つで囲む
- 引用（単一行） → `> テキスト`
- 引用（複数行） → `>>> テキスト`
- 見出し → `#` / `##` / `###`
- 箇条書き → `- リスト` / `* リスト`
- 番号付きリスト → `1. リスト`
- リンク → URL をそのまま貼る

## 使えない

- テーブル（表） — `| col |` 記法は非対応。そのままテキストとして表示される
- 画像埋め込み — `![alt](url)` は動かない
- HTML タグ

## 表の代替手段

表を使いたい場面では以下で代替する:

- 箇条書き + 太字で構造化
